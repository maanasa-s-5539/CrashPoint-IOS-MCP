# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it **privately**:

1. **Do not** open a public GitHub issue, discussion, or pull request for security vulnerabilities.
2. Use GitHub's [Private Vulnerability Reporting](https://github.com/maanasa-s-5539/CrashPoint-IOS-MCP/security/advisories/new) for this repository (Security tab → "Report a vulnerability").
3. Include a clear description, reproduction steps, affected versions, and any suggested mitigation.
4. Please allow a reasonable amount of time for the issue to be investigated and patched before any public disclosure.

We appreciate your help in keeping this project and its users secure.
